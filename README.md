# vue-music
这是使用vue仿写的网易云音乐播放器，主要使用了vue全家桶来实现的，此项目的数据请求借口来源于[Binaryify](https://github.com/Binaryify/NeteaseCloudMusicApi),所以还需将api下载至本地，安装好依赖，再使用 node app.js来启动，这样就能请求到数据

# 项目演示地址
[请点击这里](http://http://118.31.76.125)

## 项目运行
```
npm install
```
```
npm run serve
```
