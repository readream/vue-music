# vue-music
这是使用vue仿写的网易云音乐播放器，主要使用了vue全家桶来实现的，此项目的数据请求借口来源于Binaryify(https://github.com/Binaryify/NeteaseCloudMusicApi),所以还需将api下载至本地，安装好依赖，再使用 node app.js来启动，这样就能请求到数据

## Project setup
```
npm install
```
